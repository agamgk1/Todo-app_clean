Spring Todo-app
Application based on Java Spring Framework Udemy course

Application allow to create new todos, groups, projects and save it in database

Technologies:

- Java 11
- Spring 5
- Spring Boot 2
- html 5
- JUnit
- Mockito
- AssertJ
- Data
- Hibernate
- AOP
- Security
- Thymeleaf
- Flyaway
- Maven
- Dependency Injection
- Tomcat
